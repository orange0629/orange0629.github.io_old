Legend of Zelda (Remaster)

This project is finished with my groupmate Connor Young in 3 weeks.

Introduction

This is an Unity remaster version of the classic NES game "Legend of Zelda", along with a custom level design with new mechanisms. The custom level enables player to control two characters at the same time, overcome a series of puzzles, and finally defeat the boss.

Basic Control

  • Movement: WASD/Arrow Key
  • Main Weapon: X
  • Secondary Weapon: Z
  • Switch Seconday Weapon: Space
  • Switch Traditional/Custom level: 4
  • God mode: 1

    • My Contribution to Traditional level

  • UI system
  • Door mechanism
  • Items and Weapons (bomb, boomerang)
  • Room Transition
  • Pushable blocks
  • Featured rooms (bow and oldman room)

    • My Contribution to Custom level

  • Double-pushable blocks
  • Buttons
  • Room Rotations
  • Several Puzzle Designs
  • Final Boss

    • Technology used

    This project used Unity as game engine, and C# as programming language for scripts.

    Be Artist

    This project is finished independently by Lechen Zhang in 2 weeks.

    Introduction

    This is an Unity original game by Lechen Zhang. Players can paint whatever they want in this game, and all paints will become a real object. Players can use these paints as tools to get over the puzzles.

    Basic Control

  • Movement: AD/Arrow Key
  • Jump: Space
  • Draw/Erase: Mouse(Left Click)
  • Reset the level (Cheat): O
  • Next level (Cheat): P

    • My Contribution

  • Movement system
  • Paint and Physics system
  • Health system
  • Enemy system
  • Level design

    • Technology used

    This project used Unity as game engine, and C# as programming language for scripts.

    AntiVirus

    This project is finished with my groupmates in Unproductive Production (Connor Young, Kate Kim, Vincent Ma) in 4 weeks.

    Introduction

    Welcome, new employees, to Unproductive Computer Repair! You and a partner have been hired because you are some of the best tech support specialists in the business. Your mission: clean out every last bit of bugs and viruses from the computers of the masses! Use the combined power of your foam shooter and vacuum to overcome obstacles, fight enemies, and save the day!

    Antivirus is a two-player co-op Unity original game by Unproductive Production where the players take on the role of employees who work for a general-purpose IT company to take on different jobs for customers, literally traveling inside their computer to clean up viruses. You can utilize two key weapons, the foamer and the vacuum, to solve puzzles and kill enemies.

    My Contribution

  • Multiplayer Manage System
  • Controller Input System
  • Camera and Split-screen
  • Weapon System
  • Popup Menu & Task Detail Page
  • Game Progress Control System
  • SpawnPoint & CheckPoint & GoalPoint
  • Cutscene
  • Moving Platform & Springs
  • In-game UI system
  • Level Design

    • Basic Control

  • Movement: Left Stick
  • Jump: LT/A
  • Aim: Right Stick
  • Shoot: RT
  • Menu: Start

    • Technology used

    This project used Unity as game engine, and C# as programming language for scripts.

Legend of Zelda (Remaster)

This project is finished with my groupmate Connor Young

Introduction

This is an Unity remaster version of the classic NES game "Legend of Zelda", along with a custom level design with new mechanisms. The custom level enables player to control two characters at the same time, overcome a series of puzzles, and finally defeat the boss.

Basic Control

  • Movement: WASD/Arrow Key
  • Main Weapon: X
  • Secondary Weapon: Z
  • Switch Seconday Weapon: Space
  • Switch Traditional/Custom level: 4
  • God mode: 1

    • My Contribution to Traditional level

  • UI system
  • Door mechanism
  • Items and Weapons (bomb, boomerang)
  • Room Transition
  • Pushable blocks
  • Featured rooms (bow and oldman room)

    • My Contribution to Custom level

  • Double-pushable blocks
  • Buttons
  • Room Rotations
  • Several Puzzle Designs
  • Final Boss

    • Technology used

    This project used Unity as game engine, and C# as programming language for scripts.

    Be Artist

    This project is finished independently by Lechen Zhang.

    Introduction

    This is an Unity original game by Lechen Zhang. Players can paint whatever they want in this game, and all paints will become a real object. Players can use these paints as tools to get over the puzzles.

    Basic Control

  • Movement: AD/Arrow Key
  • Jump: Space
  • Draw/Erase: Mouse(Left Click)
  • Reset the level (Cheat): O
  • Next level (Cheat): P

    • My Contribution to Traditional level

  • Movement system
  • Paint and Physics system
  • Health system
  • Enemy system
  • Level design

    • Technology used

    This project used Unity as game engine, and C# as programming language for scripts.
